# 💬 Chat App - React

Aplicación web tipo chat desarrollada con React como trabajo práctico integrador.  
El objetivo del proyecto es simular el funcionamiento de una app de mensajería, enfocándose en el manejo de estado, componentes y props.

## 🌐 Demo

🔗 https://tp-react-utn-one.vercel.app/

---

## 🧠 Funcionalidades

- ✅ Crear nuevos chats dinámicamente
- ✅ Lista de contactos con estado (online/offline)
- ✅ Selección de chat activo
- ✅ Historial de mensajes independiente por chat
- ✅ Envío de mensajes
- ✅ Respuesta automática simulada
- ✅ Interfaz tipo chat (mensajes alineados)
- ✅ Diseño responsive
- ✅ Scroll automático de mensajes

---

## 🛠️ Tecnologías utilizadas

- React
- JavaScript (ES6+)
- CSS / Tailwind CSS
- Context API (manejo de estado global)
- Vite (entorno de desarrollo)
- Vercel (deploy)
